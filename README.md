
# Voice Changer

A realtime voice chnager from metasoul sotware ltd. . Using Elevenlabs voice engine Speech to Speech.



## Features

- Live Voice change
- Can clone anyone voice
- low letency
- Cross platform

To deploy this project run

```bash
  just install the exe file
```
